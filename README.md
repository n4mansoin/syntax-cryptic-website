# INTRA SYNTAX CRYPTIC

Minimalist cryptic hunt platform built with Next.js, ShadCN UI, and Tailwind CSS.

## Getting Started

1. **Setup**: Run `npm install` to install dependencies.
2. **Development**: Run `npm run dev` to start the local server.
3. **Admin Access**:
   - Login at `/admin/login`
   - Default credentials: `admin` / `qawsedrftg`
4. **Hunt initialization**:
   - Visit `/admin/setup` to restore factory levels and teams.

## Pushing to Git

```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin <your-repo-url>
git branch -M main
git push -u origin main
```

## Tech Stack
- Next.js (App Router)
- Tailwind CSS
- ShadCN UI
- Lucide Icons
- GSAP (Animations)
- BroadcastChannel (Real-time Sync)
